# stefan_denisa
Social Media Scraper - Am extras date din platforma Reddit - postări ce includ cuvântul-cheie Romania. Am folosit modulul RSS din Make și am centralizat datele în Google Sheets. Integrarea rulează o dată la 2 ore și generează maxim 10 postări din subreddit-ul r/romania ce conțin cuvântul-cheie Romania.
